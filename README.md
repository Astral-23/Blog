# Markdown Blog Framework

このリポジトリの入口です。  
詳細は `docs/` を見てください。

## どれを読めばいいか

- 運用者（記事を書く人）: `docs/admin-guide.md` だけ読めばOK
- 開発者（機能を直す人）: `docs/developer-guide.md`
- Codexへ公開前チェック依頼: `docs/codex-prepublish-check.md`

## 最短コマンド

```bash
npm install
npm run dev
```

公開前:

```bash
npm run publish:secure
```
